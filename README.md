# Dietregistrering - Mässrestauranger

Ett webbaserat formulär för att registrera specialkost och allergier för eventdeltagare.

## Funktioner

- **Företags- och eventinformation** - Registrera kontaktuppgifter och eventdetaljer
- **Deltagare med dietbehov** - Lägg till obegränsat antal deltagare
- **Flexibel dag/måltidsval** - Välj vilka dagar och måltider varje deltagare ska ha
- **Kostpreferenser** - Vegetarisk, Vegansk, Pescetarian, Gravid, IBS, Halal
- **Allergener** - Alla 14 deklarationspliktiga allergener enligt Livsmedelsverket
- **Flerspråkigt** - Svenska och engelska
- **Export** - PDF och Excel med valbart språk

## Allergener som stöds

Gluten, Laktos, Mjölkprotein, Baljväxter, Kräftdjur, Ägg, Fisk, Jordnötter, Lupin, Soja, Nötter, Stenfrukt, Selleri, Senap, Sesamfrön, Svaveldioxid/Sulfit, Blötdjur

## Användning

1. Fyll i företags- och eventinformation
2. Lägg till deltagare med dietbehov via "+ Lägg till deltagare"
3. Välj dagar och måltider för varje deltagare
4. Ange kostpreferenser och allergier
5. Klicka "Spara och granska registrering"
6. Välj exportspråk och ladda ner PDF eller Excel

## Export

- **PDF** - Översiktlig lista för utskrift
- **Excel** - Detaljerade flikar per dag + sammanställningsflik, perfekt för köket

## Teknisk information

Ren HTML/CSS/JavaScript utan beroenden på server. Använder:
- [ExcelJS](https://github.com/exceljs/exceljs) för Excel-export
- [jsPDF](https://github.com/parallax/jsPDF) för PDF-export
